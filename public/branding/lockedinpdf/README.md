# LockedInPDF logo package

Production SVG sources for the approved browser, focus, document, and lock concept.

## Files

- `mark-light.svg`: full-color mark for light backgrounds
- `mark-dark.svg`: full-color mark for dark backgrounds
- `mark-mono.svg`: monochrome mark
- `lockup-light.svg`: horizontal mark and wordmark for light backgrounds
- `lockup-dark.svg`: horizontal mark and wordmark for dark backgrounds
- `favicon-source.svg`: simplified dark app-icon source

## Brand colors

- Midnight navy: `#071A3D`
- Cobalt blue: `#1769E8`
- Dark-mode cobalt: `#4D86FF`
- Emerald: `#12A567`
- Dark-mode emerald: `#24C47E`
- Off-white: `#F7FAFF`

These files are staged separately from the current site assets. Replace the active favicon, PWA, desktop, and header assets only when the LockedInPDF name is cleared and the full rebrand is approved.
